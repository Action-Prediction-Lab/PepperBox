# Resources

PepperBox does not ship Aldebaran-authored printable assets. The official
NAOmark printable sheet is hosted by Aldebaran at:

- <http://doc.aldebaran.com/2-5/_downloads/NAOmark.pdf>

If that URL is unreachable, an Internet Archive snapshot is at:

- <https://web.archive.org/web/20231222033304/http://doc.aldebaran.com/2-5/_downloads/NAOmark.pdf>

The relevant API documentation (`ALLandMarkDetection`) lives at
<http://doc.aldebaran.com/2-5/naoqi/vision/allandmarkdetection.html>.
